Important FAQ

What is the Document Object Model(DOM)?
The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document (like HTML or XML) as a tree of objects, allowing scripts (usually JavaScript) to manipulate the content, structure, and style of a webpage dynamically.

How do you select an element from the DOM?
getElementById: Selects an element by its unique ID.getElementsByClassName: Selects all elements with a specific class name. This returns a live HTMLCollection.querySelector: Selects the first element that matches a specified CSS selector.querySelectorAll: Selects all elements that match a specified CSS selector. This returns a static NodeList.

What is event delegation in the context ot the DOM, and why is it useful?
Event delegation is a technique in the DOM where a single event listener is attached to a parent element instead of adding individual listeners to multiple child elements. This listener handles events for all child elements, even those added dynamically later.

How do you manipulate an elment's Attributes and styles using the DOM?
Manipulating Attributes Get Attribute: Use getAttribute('attributeName') to retrieve an attribute's value. Set Attribute: Use setAttribute('attributeName', 'value') to set or update an attribute. Remove Attribute: Use removeAttribute('attributeName') to delete an attribute. Access Properties: Some attributes can be accessed directly as properties (e.g., element.id, element.className). Manipulating Styles Inline Styles: Use element.style.property = 'value' to set individual CSS styles (e.g., element.style.color = 'red'). Multiple Styles: Use element.style.cssText = 'property: value;' to apply multiple styles at once. Class Management: Use element.classList to add, remove, or toggle classes (e.g., element.classList.add('active')).
